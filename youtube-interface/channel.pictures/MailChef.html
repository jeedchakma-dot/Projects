🚀 Setup guide — run MailChef

This HTML file is your complete frontend. To enable real email sending, customer syncing, and data storage you also need to run the backend server.

1
Download the backend — click the button below to get setup-mailchef-backend.sh
2
Run the setup script — it creates all backend files automatically:
bash setup-mailchef-backend.sh cd mailchef-backend npm install
3
Create a .env file in the backend folder:
ANTHROPIC_API_KEY=sk-ant-your-key-here RESEND_API_KEY=re_your-key-here JWT_SECRET=any-long-random-string PORT=4000
Get keys free at console.anthropic.com and resend.com
4
Start the server:
node server.js
You'll see: 🕯 MailChef SaaS running on http://localhost:4000
5
Open MailChef.html in your browser — register an account, connect a data source, and you're live.
6
To go fully live online: Deploy the backend folder to Railway.app or Render.com (both free to start). Then open MailChef.html in a text editor, find localhost:4000 and replace with your live URL.
